XP Image Tools 소개

XP Image Tools는 Windows XP 스타일의 인터페이스로 구성된 무료 온라인 이미지 편집 도구입니다. 이미지 크기 변경, 압축, PNG/JPG/WEBP 형식 변환 등 다양한 기능을 브라우저에서 직접 처리합니다. 별도 프로그램 설치 없이 웹 브라우저만 있으면 즉시 사용할 수 있습니다.

이미지 크기 변경 (리사이즈)

이미지의 가로·세로 픽셀을 자유롭게 조절합니다. 비율 유지 옵션으로 한쪽 값만 입력해도 나머지가 자동 계산됩니다. FHD(1920×1080), HD(1280×720), VGA(640×480) 등 빠른 크기 선택을 지원합니다. 단계적 리샘플링으로 화질 저하를 최소화합니다.

이미지 압축

JPG, PNG, WEBP 이미지의 파일 크기를 줄입니다. 자동 모드에서는 목표 파일 크기(50KB~1MB)를 입력하면 이진 탐색 알고리즘이 최적 품질을 자동으로 찾습니다. 수동 모드에서는 품질 슬라이더로 직접 조절합니다. 이메일 첨부, SNS 업로드, 웹 최적화에 활용하세요.

이미지 형식 변환

PNG, JPG, WEBP 세 가지 형식을 서로 자유롭게 변환합니다. PNG는 투명 배경 지원 무손실 포맷, JPG는 사진에 최적화된 손실 압축 포맷, WEBP는 JPG 대비 25~35% 작은 최신 포맷입니다.

개인정보 보호

모든 이미지 처리는 사용자 브라우저에서 직접 이루어지며, 서버로 전송되지 않습니다. 회원가입, 로그인 없이 바로 사용 가능합니다. 업로드한 이미지는 브라우저 탭을 닫으면 완전 소멸됩니다.
