📝 이메일 작성 가이드

빠른 답변을 위해 이메일 작성 시 아래 내용을 포함해 주시면 도움이 됩니다.

  • 문의 유형: 버그 신고 / 기능 제안 / 사용법 문의 / 기타
  • 사용 환경: 브라우저 종류 및 버전, 운영체제(Windows/Mac/Android/iOS)
  • 문제 상황: 어떤 기능을 사용하다가 어떤 문제가 발생했는지 구체적으로 설명
  • 재현 방법: 문제를 재현할 수 있는 단계별 설명
  • 스크린샷: 오류 화면이나 문제 상황 캡처 이미지 (선택)

📌 참고: 자주 묻는 질문은 FAQ 페이지를 먼저 확인해 주세요. 대부분의 질문에 빠른 답변이 있습니다.

⚠️ 문의 전 확인사항

  • 이미지가 업로드되지 않는 경우 → 브라우저를 최신 버전으로 업데이트해 주세요.
  • 처리 결과가 이상한 경우 → 다른 브라우저에서 시도해 주세요 (Chrome 권장).
  • 다운로드가 안 되는 경우 → 브라우저의 팝업 차단 설정을 확인해 주세요.
  • 처리 속도가 너무 느린 경우 → 파일 크기가 30MB 이하인지 확인해 주세요.

🐛 버그 신고 시 포함할 정보

버그를 신고하실 때 아래 정보를 포함해 주시면 훨씬 빠르게 해결할 수 있습니다.

  • 브라우저: Chrome 버전 몇인지, Firefox인지, Safari인지
  • 운영체제: Windows 11, macOS, Android, iOS 등
  • 사용한 기능: 크기 변경, 압축, 형식 변환 중 어느 기능
  • 이미지 형식: JPG, PNG, WEBP 중 어떤 파일을 사용했는지
  • 오류 메시지: 화면에 표시된 오류 메시지 전체 내용
  • 재현 방법: 몇 번 시도했을 때 오류가 발생하는지
  • 스크린샷: 가능하다면 오류 화면 캡처 첨부
