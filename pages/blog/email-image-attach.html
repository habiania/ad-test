이메일 첨부 이미지 용량 줄이기 — 10MB를 200KB로 만드는 법

📅 2025-06-07 ⏱️ 읽는 시간 약 4분 ✍️ XP Image Tools 팀

스마트폰으로 찍은 사진을 이메일로 보내려고 하면 "첨부 파일 크기 제한 초과" 오류가 뜨는 경험을 해보셨나요? 최신 스마트폰 사진은 한 장에 5~15MB에 달해 이메일 첨부에 적합하지 않습니다. 몇 번의 클릭으로 10MB 사진을 200KB로 줄이는 방법을 알아봅니다.

사진 여러 장을 함께 보내야 한다면 각 파일을 더 작게 압축해야 합니다. 5장을 보낼 경우 각 파일을 500KB 이하로 유지하면 대부분의 서비스에서 문제없이 전송됩니다.

🛠️ XP Image Tools로 이메일용 사진 만들기

총 3단계로 이메일에 적합한 사진을 만들 수 있습니다.

방법 1: 자동 압축 (가장 쉬움)

  1. XP Image Tools 접속 → 「📦 이미지 압축」 선택
  2. 이메일로 보낼 사진 업로드
  3. 자동 모드에서 목표 크기 「200KB」 선택
  4. 출력 형식 「JPG」 선택
  5. 「압축 실행」 → 결과 확인 후 다운로드

방법 2: 크기 축소 후 압축 (화질 중시)

원본이 4000px 이상의 고해상도라면 먼저 크기를 줄인 후 압축하면 더 좋은 결과를 얻을 수 있습니다.

  1. 「🖼️ 이미지 크기 변경」에서 가로를 1600px로 줄이기 (비율 유지 체크)
  2. JPG 형식, 품질 85%로 다운로드
  3. 결과 파일이 여전히 크다면 「📦 이미지 압축」에서 추가 압축

✅ 결과 예시: 12MB(4000×3000px) 스마트폰 사진 → 크기 1600×1200px 조절 + 자동 압축 200KB → 최종 195KB. 화질은 이메일 확인에 충분한 수준 유지.

💡 이메일 이미지 전송 시 알아두면 좋은 것들

  • 파일명에 한글 지양: 일부 이메일 클라이언트에서 한글 파일명이 깨질 수 있습니다. 영문 파일명을 사용하는 것이 안전합니다.
  • 여러 장은 ZIP으로 묶기: 사진 10장 이상은 ZIP으로 압축하여 하나의 파일로 전송하면 받는 사람이 편리합니다.
  • 고화질이 필요하다면 클라우드 링크: 인쇄용 원본이 필요한 경우 구글 드라이브나 카카오드라이브 링크를 공유하는 것이 낫습니다.

🔐 업무용 이미지 전송 시 보안 주의사항

업무상 기밀이 포함된 이미지(계약서, 내부 문서, 개인정보 등)를 이메일로 전송할 때는 보안에 주의해야 합니다. XP Image Tools는 모든 처리가 브라우저에서만 이루어지므로 이미지 데이터가 외부 서버로 전송되지 않습니다. 따라서 기밀 문서의 스크린샷도 안심하고 크기를 줄일 수 있습니다. 단, 이메일 전송 자체는 수신자의 보안 환경에 따라 달라지므로 매우 민감한 정보는 암호화된 메일이나 보안 문서 공유 서비스를 사용하는 것을 권장합니다.

📱 스마트폰에서 바로 하는 법

스마트폰으로 찍은 사진을 바로 XP Image Tools로 처리할 수 있습니다. 모바일 Chrome이나 Safari 브라우저에서 XP Image Tools에 접속한 후, 파일 선택 버튼을 누르면 갤러리에서 직접 사진을 선택할 수 있습니다. 압축 후 다운로드된 파일은 갤러리에 저장되며, 이메일 앱에서 바로 첨부할 수 있습니다.

⚡ 자주 쓰는 설정 빠르게 적용하기

이메일 업무가 많다면 자주 사용하는 설정을 기억해 두세요. 일반 업무 사진은 자동 압축 200KB, 출력 형식 JPG로 설정하면 대부분의 상황에서 적합합니다. 스크린샷이나 문서 이미지는 PNG 형식으로 저장해야 텍스트가 선명하게 유지됩니다. 이미지에 텍스트가 포함된 경우 JPG로 압축하면 텍스트 주변이 흐릿해질 수 있습니다.
