📖 사용 가이드

XP Image Tools의 각 기능을 단계별로 쉽게 따라할 수 있도록 안내합니다.

💡 핵심 원칙: 모든 이미지 처리는 여러분의 브라우저에서 직접 실행됩니다. 이미지가 서버로 전송되지 않으므로 안심하고 사용하세요.

세 가지 도구(크기 변경·압축·형식 변환) 모두 동일한 방식으로 이미지를 입력합니다.

🖼️ 이미지 크기 변경 가이드

이미지의 가로·세로 픽셀을 원하는 크기로 조절합니다. SNS 업로드 규격, 이메일 첨부, 웹사이트 이미지 최적화 등에 활용하세요.

  • 홈 화면의 시작 메뉴에서 「🖼 이미지 크기 변경」을 클릭하거나, 데스크탑의 이미지 아이콘을 더블클릭합니다.
  • 이미지를 업로드합니다. 원본 이미지의 가로·세로 픽셀과 파일 크기가 표시됩니다.
  • 가로(W) 또는 세로(H) 값을 입력합니다. 「비율 유지」 체크 시 한쪽만 입력해도 나머지가 자동 계산됩니다.
  • 빠른 크기 선택 버튼(FHD·HD·800·VGA·200²)으로 자주 쓰는 크기를 한 번에 선택할 수도 있습니다.
  • 출력 형식(원본 유지·JPG·PNG·WEBP)을 선택하고, JPG·WEBP는 품질 슬라이더를 조절합니다.
  • 「⚡ 크기 변경 실행」 버튼을 클릭합니다. 결과 미리보기가 우측에 나타납니다.
  • 「💾 다운로드」 버튼을 클릭하면 원본파일명-resized-가로x세로.확장자 형식으로 저장됩니다.

✅ 화질 팁: 크게 축소할 때는 PNG 출력이 가장 선명합니다. JPG는 품질 85% 이상이면 육안으로 구분하기 어렵습니다.

📦 이미지 압축 가이드

파일 크기를 줄이되 화질은 최대한 유지합니다. 이메일 첨부 용량 제한, SNS 파일 크기 제한, 웹사이트 로딩 속도 개선에 효과적입니다.

  • 시작 메뉴에서 「📦 이미지 압축」을 선택합니다.
  • 이미지를 업로드합니다. 원본 파일 크기가 표시됩니다.
  • 압축 방식을 선택합니다.
    • 자동 모드: 목표 크기(50KB / 100KB / 200KB / 500KB / 1MB)를 선택합니다. 알고리즘이 자동으로 최적 품질을 찾아줍니다.
    • 수동 모드: 품질 슬라이더(10~99%)를 직접 조절합니다.
  • 출력 형식(JPG·WEBP·PNG)을 선택합니다. WEBP가 가장 높은 압축률을 제공합니다.
  • 「⚡ 압축 실행」을 클릭합니다. 우측에 압축된 미리보기와 절감률(%)이 표시됩니다.
  • 만족스러우면 「💾 다운로드」로 저장합니다.

✅ 압축 팁: 사진 파일은 JPG 품질 75~85%, 웹 이미지는 WEBP 형식이 크기 대비 화질이 가장 우수합니다.

🔄 이미지 형식 변환 가이드

PNG ↔ JPG ↔ WEBP를 자유롭게 변환합니다. 각 형식의 특성에 맞게 목적에 따라 선택하세요.

형식특징권장 용도
JPG손실 압축 · 파일 작음 · 투명 배경 미지원사진, 블로그 이미지
PNG무손실 압축 · 투명 배경 지원 · 파일 큼로고, 아이콘, UI 요소
WEBP최신 포맷 · 가장 작음 · 투명 배경 지원웹사이트 이미지, SNS
  • 시작 메뉴에서 변환할 형식(PNG 변환 / JPG 변환 / WEBP 변환)을 선택하거나, 변환 도구에서 상단 탭으로 변경합니다.
  • 이미지를 업로드합니다. 원본 형식·크기가 표시됩니다.
  • JPG·WEBP 출력 시 품질 슬라이더를 조절합니다(PNG는 무손실이라 별도 설정 없음).
  • 「⚡ 변환 실행」을 클릭합니다. 우측에 변환 결과 미리보기와 크기 변화가 표시됩니다.
  • 「💾 다운로드」로 저장합니다. 파일명은 원본파일명.변환형식 으로 자동 생성됩니다.

⚠️ 주의: JPG로 변환 시 투명 배경(알파 채널)은 흰색으로 채워집니다. 투명 배경을 유지해야 한다면 PNG 또는 WEBP로 변환하세요.

📧 이메일 첨부 팁

대부분의 이메일 서비스는 첨부 파일 합계 25MB 이하를 권장합니다. 사진 여러 장을 보낼 때는 자동 압축 모드에서 각 파일을 200~500KB로 압축하면 빠르게 전송할 수 있습니다.

🌐 웹사이트 이미지 최적화 팁

웹사이트 로딩 속도는 SEO와 사용자 경험 모두에 영향을 줍니다. 웹 이미지는 WEBP 형식으로 변환하고, 파일 크기는 100~300KB 이하로 유지하는 것을 권장합니다. 배경 이미지나 히어로 이미지는 1920px, 일반 본문 이미지는 1200px 이하로 크기를 조절하세요.

🖨️ 인쇄용 이미지 팁

인쇄용 이미지는 해상도보다 픽셀 크기가 중요합니다. A4 사이즈(300DPI 기준)에서는 2480×3508px, 명함 크기에서는 1063×591px이 필요합니다. PNG 무손실 형식으로 저장하면 화질 손실 없이 인쇄할 수 있습니다.

📱 실전 사용 시나리오

시나리오 1: 인스타그램 정방형 사진 만들기

인스타그램 피드에 올릴 정방형(1:1 비율) 이미지를 만드는 방법입니다. 이미지 크기 변경 도구에서 비율 유지를 해제하고 가로 1080, 세로 1080을 입력합니다. 출력 형식은 JPG를 선택하고 품질은 90%로 설정합니다. 이렇게 하면 인스타그램 권장 해상도인 1080×1080px 이미지가 완성됩니다.

시나리오 2: 이메일 첨부용 사진 압축하기

회사 이메일 서버가 첨부 파일을 10MB로 제한하는 경우를 예로 들겠습니다. 스마트폰으로 찍은 10MB짜리 사진을 이메일에 첨부하려면 압축이 필요합니다. 압축 도구에서 자동 모드를 선택하고 목표 크기를 500KB로 설정합니다. 실행하면 약 95% 이상 용량이 줄어들면서도 충분히 선명한 사진이 유지됩니다.

시나리오 3: 웹사이트용 WEBP 이미지 만들기

워드프레스나 개인 홈페이지에 올릴 이미지를 WEBP로 변환하면 페이지 로딩 속도를 크게 개선할 수 있습니다. 형식 변환 도구에서 WEBP를 선택하고 품질 85%로 설정합니다. 일반적으로 JPG 대비 25~35% 파일 크기가 줄어들어 SEO 점수 향상에도 도움이 됩니다.

시나리오 4: 발표 자료용 PNG 로고 만들기

PowerPoint나 Keynote에 삽입할 투명 배경 로고가 필요한 경우, PNG 형식으로 변환하면 됩니다. 형식 변환 도구에서 PNG를 선택하면 투명 배경이 그대로 유지됩니다. JPG로 변환하면 투명 부분이 흰색으로 채워지므로 로고는 반드시 PNG로 저장하세요.

⌨️ 단축키 및 편의 기능

  • Ctrl+V (클립보드 붙여넣기): 도구 패널이 열려 있는 상태에서 스크린샷이나 복사한 이미지를 바로 붙여넣기할 수 있습니다.
  • 드래그 앤 드롭: 파일 탐색기에서 이미지를 점선 박스로 드래그하면 즉시 로드됩니다.
  • 초기화 버튼: 새 이미지를 처리하려면 초기화 버튼을 클릭하여 처음 상태로 돌아가세요.
  • 시작 메뉴: 화면 좌하단의 XP 스타일 시작 버튼을 클릭하면 모든 도구에 빠르게 접근할 수 있습니다.

🔧 문제 해결

이미지가 업로드되지 않을 때

  • 지원 형식(JPG, PNG, WEBP)인지 확인하세요.
  • 브라우저를 최신 버전으로 업데이트하세요.
  • Chrome 또는 Firefox 사용을 권장합니다.
  • 파일이 30MB 이하인지 확인하세요.

다운로드가 되지 않을 때

  • 브라우저의 팝업 차단 설정을 확인하고 이 사이트의 팝업을 허용해 주세요.
  • 주소창 옆의 차단 아이콘을 클릭하여 허용으로 변경하세요.

처리 결과가 예상과 다를 때

  • 비율 유지 옵션 체크 여부를 확인하세요.
  • 출력 형식과 품질 설정을 다시 확인하세요.
  • 문제가 지속되면 문의 페이지로 알려주세요.
